# Lumix University - Event Management System

## Project Overview
The **Lumix University Event Management System** is a modern, cross-platform Progressive Web Application (PWA) designed to enhance student engagement. Built on Microsoft's **Blazor WebAssembly (WASM)** and integrated with **Firebase**, it provides a seamless, real-time experience for students to discover and participate in campus activities.

## Key Features
- **Event Discovery:** Browse upcoming and ongoing campus events with real-time fuzzy search and category filtering.
- **Instant RSVP:** Quick registration for events with automatic validation and real-time cloud synchronization.
- **Attendee Feedback:** Secure post-event feedback system with numeric ratings and qualitative reviews.
- **Admin Dashboard:** Centralized panel for managing events (CRUD) and monitoring real-time student engagement analytics.
- **Native Experience (PWA):** Installable on both mobile and desktop devices with offline caching support.
- **Responsive Design:** Optimized for Mobile (iPhone), Tablet (iPad Air), and High-DPI Desktop viewports using Bootstrap 5.

## Technology Stack
- **Frontend Framework:** .NET 8.0 Blazor WebAssembly
- **Cloud Infrastructure:** Firebase Cloud Firestore (NoSQL)
- **Styling:** CSS3, Bootstrap 5, Modern Typography
- **Interoperability:** `IJSRuntime` for high-performance C# to JavaScript SDK communication.
- **Architecture:** MVVM-pattern with Service-Oriented State Management.

## Prerequisites
- [.NET 8.0 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
- [Node.js](https://nodejs.org/) (for Firebase CLI if needed for deployment)
- Modern Web Browser (Chrome, Edge, or Safari)

## Installation and Locally Running
1. **Clone/Open the Project:**
   Navigate to the project root directory.
2. **Restore Dependencies:**
   ```bash
   dotnet restore
   ```
3. **Run the Application:**
   ```bash
   dotnet run --project LumixApp
   ```
4. **Access the Application:**
   Open your browser and navigate to `http://localhost:5245`.
